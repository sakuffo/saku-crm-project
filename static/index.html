CRM Backend API

This API should be a straightforward stub to work with customers in a mock CRM database. The Database is a map[string]Customer, where Customer is a struct constructed to manage customers. NB: App is on port 3300 as port 3000 is airplay port on macOS

API Paths

    Get a Single Customer

    HTTP VERB: GET

    "http://{host-address:port}/customer/{id}" to pull a customer.

    e.g. (when runing locally) "http://localhost:3300/customer/5"

    Get a All Customer

    HTTP VERB: GET

    "http://{host-address:port}/customers/" to pull all customer.

    e.g. (when runing locally) "http://localhost:3300/customers/"

    Creating a customer

    HTTP VERB: POST

    "http://{host-address:port}/customers/" w/ JSON to create customer.

    Submit a json payload. ID not required e.g.

    { "Name": "Jeff Rider", "Role": "Help Desk", "Email": "JeffR@admin.org", "PhoneNumber": "676-767-6677", "Contacted": false }

    Updating a customer

    HTTP VERB: PATCH

    "http://{host-address:port}/customers/" w/ JSON to update customer.

    Submit a json payload. ID required to select record. e.g.

    { "Id": 1, "Name": "Stephen Kin", "Role": "IT Director", "Email": "skin@admin.org", "PhoneNumber": "677-787-1009", "Contacted": true }

    Delete a Customer

    HTTP VERB: DELETE

    "http://{host-address:port}/customer/{id}" to delete a customer.

    e.g. (when runing locally) "http://localhost:3300/customer/5"
